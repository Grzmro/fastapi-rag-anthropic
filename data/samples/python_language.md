# The Python Programming Language

## Origins

Python was created by Guido van Rossum and first released in 1991. Van Rossum
began working on it in December 1989 as a hobby project during the Christmas
holidays. The language was named after the British comedy group Monty Python,
not the snake. Van Rossum served as Python's "Benevolent Dictator For Life"
(BDFL) until he stepped down from that role in 2018.

## Design philosophy

Python emphasizes code readability and simplicity. Its guiding principles are
captured in "The Zen of Python", a collection of 19 aphorisms written by Tim
Peters, accessible by running `import this`. A distinctive feature of the
language is that it uses indentation to define code blocks instead of curly
braces.

## Major versions

Python 2.0 was released in 2000 and introduced list comprehensions and a
garbage collector. Python 3.0, released in 2008, was a major backwards-
incompatible revision that fixed several long-standing design flaws. Python 2
reached its official end of life on January 1, 2020, meaning it no longer
receives security updates.

## Ecosystem

Python has a large standard library, often described as "batteries included".
The Python Package Index (PyPI) hosts hundreds of thousands of third-party
packages. Popular libraries include NumPy and pandas for data analysis,
Django and FastAPI for web development, and PyTorch and TensorFlow for machine
learning. The reference implementation of the language is called CPython and
is written in C.
